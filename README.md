# セイカクブック - あなたにピッタリな本をお勧め!

MBTI（Myers-Briggs Type Indicator）性格診断を行い、診断結果に基づいてあなたにぴったりの本を推薦するWebアプリケーションです。

## 主な機能

### 📖 3つの入力方法
1. **MBTIタイプを直接入力**: 既に知っているMBTIタイプを選択して本を推薦
2. **Strength Finderの結果を入力**: Strength Finderの上位5つの強みを入力してMBTIタイプを推測し、本を推薦
3. **性格診断をする**: 16の質問に答えてMBTIタイプを診断し、本を推薦

### 📚 My Books機能
- **本の保存**: 推薦された本を「保存する」ボタンでリストに追加
- **読書ステータス管理**: 「読みたい」「読んでる」「読んだ！」の3つの状態で管理
- **星評価**: 「読んだ！」状態の本に1〜5つ星で評価を付けられる
- **書評機能**: 「読んだ！」状態の本についての感想や書評を記録できる
- **統計表示**: 各状態の本の数を一目で確認
- **ヘッダーからアクセス**: どの画面からでもMy Booksリストにアクセス可能

### ⚙️ 設定機能
- **Amazonアフィリエイト設定**: Amazonアフィリエイトタグを設定すると、すべてのAmazonリンクに自動的にタグが追加されます

### 🎯 その他の機能
- **16種類の性格タイプ判定**: INTJ、INTP、ENTJ、ENTP、INFJ、INFP、ENFJ、ENFP、ISTJ、ISFJ、ESTJ、ESFJ、ISTP、ISFP、ESTP、ESFP
- **タイプ別の本の推薦**: 各MBTIタイプに最適な3冊の本を推薦
- **Amazonリンク生成**: 推薦された本のAmazon検索リンクを自動生成
- **Strength Finderマッピング**: Strength Finderの34の強みからMBTIタイプを推測
- **ローカルストレージ保存**: ブラウザのローカルストレージにデータを自動保存（ページをリロードしても残る）

## 使い方

### 1. アプリを開く
`index.html`をブラウザで開きます。

### 2. 性格診断またはタイプ入力
ホーム画面から以下のいずれかを選択：

- **MBTIタイプを入力**: ドロップダウンからタイプを選択
- **Strength Finderの結果を入力**: 上位5つの強みを入力
- **性格診断をする**: 16の質問に答える

### 3. 推薦された本を確認
診断結果に基づいて、あなたのMBTIタイプに最適な3冊の本が表示されます。

### 4. 本を保存
気に入った本の「保存する」ボタンをクリックすると、My Booksリストに追加されます。

### 5. My Booksリストで管理
ヘッダーの「My Books」ボタンをクリックして、保存した本のリストを開きます。

- **統計を確認**: 各状態（読みたい、読んでる、読んだ！）の本の数が表示されます
- **本を選択**: リストから本をクリックして詳細画面を開きます
- **読書ステータスを変更**: 「読みたい」「読んでる」「読んだ！」から選択
- **評価と書評を追加**: 「読んだ！」状態の本に星評価と書評を追加できます

### 6. Amazonアフィリエイト設定（オプション）
ヘッダーの「設定」ボタンをクリックして、Amazonアフィリエイトタグを入力・保存します。
設定後、すべてのAmazonリンクに自動的にタグが追加されます。

### 7. 本を購入
推薦された本や保存した本の「Amazonで見る」リンクから購入できます。

## ブラウザ対応

モダンブラウザ（Chrome、Firefox、Safari、Edge）で動作します。

## データの保存について

すべてのデータ（保存した本、評価、書評、設定）はブラウザのローカルストレージに保存されます。
- ページをリロードしてもデータは残ります
- ブラウザのキャッシュをクリアするとデータが削除されます
- 別のブラウザやデバイスではデータは共有されません
